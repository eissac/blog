uniform hashing assumption

均匀分布 each key is equally likely to hash to an integer between 0-M-1

hash code冲突概率
birthday problem 跟号~piM/2 个key后非常可能有重复的hashcode

coupon collector 每个hash code都至少冲突一次 ~MlnM个key

bug
